# Summary

A simple 3d printable slider that can be used for any application that requires a linear sled.

# Print Settings

Printer Brand: MakerBot
Printer: MakerBot Replicator 2
Rafts: No
Supports: No
Resolution: 0.3
Infill: 10%

Notes: 
You may want to use 0.2 resolution for the threaded parts and the inserts at the bottom.
